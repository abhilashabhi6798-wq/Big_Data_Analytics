# Exercise 8 - Apache Cassandra

## Aim

To install Apache Cassandra and perform CRUD operations using Cassandra Query Language (CQL).

## Software Requirements

- Ubuntu WSL
- Docker
- Apache Cassandra
- CQLSH

## Files

- docker-compose.yml
- Cassandra CQL Commands

## Operations Performed

- Created Keyspace
- Created Tables
- Inserted Sample Records
- Retrieved Data
- Executed Required Queries

## Tables Created

1. song_info_by_session
2. song_playing_history_by_user
3. who_listened_to_song

## Queries Executed

### Query 1

Display all songs played in Session 100.

### Query 2

Display all songs played by User 1 during Session 100.

### Query 3

Display all users who listened to the song "Hey Jude".

## Result

Apache Cassandra was successfully installed and configured. All tables were created, records inserted successfully, and the required queries executed successfully.
