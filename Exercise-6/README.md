# Experiment 6: Weather Data Analysis using Hadoop Streaming

## Aim
To analyze weather data using Hadoop Streaming and find the maximum temperature.

## Objective
- To understand Hadoop Streaming.
- To implement Mapper and Reducer programs in Python.
- To process weather data using Hadoop MapReduce.
- To identify the maximum temperature.

## Software Used
- Ubuntu 24.04
- Hadoop 3.4.1
- Python 3
- Java 11

## Input
A weather dataset containing date, city, temperature and humidity.

## Output
The program displays the maximum temperature from the weather dataset.

## Files
- mapper_max_temp.py
- reducer_max_temp.py
- weather_data.txt
- weather_large.txt

## Result
The Hadoop Streaming job was executed successfully and the maximum temperature was obtained.
