# Experiment 7: Top Products using Hadoop Streaming

## Aim
To analyze sales data using Hadoop Streaming and identify the top five products based on total sales.

## Objective
- To understand Hadoop Streaming.
- To implement Mapper and Reducer programs in Python.
- To calculate the total sales for each product.
- To display the top five products based on sales.

## Software Used
- Ubuntu 24.04
- Hadoop 3.4.1
- Python 3
- Java 11

## Input
A sales dataset containing:
- Date
- Product
- Price
- City

## Output
The program displays the top five products based on total sales.

## Files
- generate_sales_products.py
- mapper_top_products.py
- reducer_top_products.py
- sales_products_large.txt
- top_products_output.txt

## Result
The Hadoop Streaming job was executed successfully, and the top five products based on total sales were obtained.
